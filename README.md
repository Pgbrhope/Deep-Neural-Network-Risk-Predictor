🔹 Project Overview

This project develops a predictive model using Deep Neural Networks (DNN) to identify high-potential targets based on structured features and text embeddings.

The model is designed to support decision-making by providing risk prediction and early signal detection, which can be extended to real-world applications such as public health monitoring and outbreak prediction.

🔹 Methods
Deep Neural Networks (DNN)
Transformer-based embeddings (RoBERTa)
Hyperparameter optimization (Optuna)
Imbalanced data handling (SMOTE / SMOTE-ENN)
Feature importance analysis (XGBoost + SHAP)
🔹 Model Architecture
Input:
Structured features (14 features)
Text embeddings (100-dimensional)
Processing:
Feature fusion using Dense layers
Multi-layer DNN architecture
Regularization techniques (Dropout, etc.)
Output:
Binary classification (prediction result)
